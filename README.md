### 🧭 Sekilas tentang saya

Saya seorang **Full Stack Developer** yang senang bergerak di dua dunia sekaligus — mobile dan web. Sehari-hari bergelut dengan Flutter untuk aplikasi mobile, Laravel untuk backend yang solid, dan Next.js untuk antarmuka web yang responsif. Belakangan ini sedang memperdalam pemahaman tentang cloud computing, system design, dan microservices — karena membangun aplikasi yang scalable bukan cuma soal kode yang jalan, tapi soal kode yang tahan banting.
